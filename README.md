# 梦幻拖拉机小程序原型

这是一个可继续开发的多人“梦幻拖拉机”工程骨架，包含：

- `miniprogram/`：微信小程序页面原型
- `server/`：Node.js WebSocket 房间服务雏形
- `shared/`：牌、牌型、结算等共享规则逻辑

## 第一版规则

- 支持 2-10 人房间。
- 1 副 54 张牌，可配置大小王是否作为万能牌。
- 第一局房主当庄，后续按座位顺时针或逆时针轮庄。
- 每人 1 张暗牌，桌面 1 张公共牌。
- 每个玩家可用自己的暗牌、公共牌，以及幻想出来的第三张牌组成最大三张牌型。
- 闲家依次叫分。
- 庄家可选择开单人、通开所有人、不开。
- 小程序内使用“叫分/惩罚值”表达，不直接绑定线下饮酒。

## 运行后端

```bash
npm install
npm run server:dev
```

默认 WebSocket 地址：

```text
ws://localhost:8787
```

## 小程序预览

用微信开发者工具导入 `miniprogram/` 目录即可。当前前端先使用本地模拟牌局数据，后续可以把 `utils/socket.js` 接到后端 WebSocket。
